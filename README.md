# EDFViewerMonitor

EDFViewerMonitor is a lightweight desktop utility for EDFViewer NEXT. It reads binary telemetry packets over USB/UART, displays live sensor data, and can record or replay binary logs.

## Build the EXE

```bash
python build_exe.py
```

The EXE will be placed in `dist/EDFViewerMonitor.exe`.
